> File system watcher for Cubomedia


## Install

```sh
$ npm install --save cubomedia-watcher
```


## Usage

```js
var cubomediaWatcher = require('cubomedia-watcher');

cubomediaWatcher('Rainbow');
```

```sh
$ npm install --global cubomedia-watcher
$ cubomedia-watcher --help
```
